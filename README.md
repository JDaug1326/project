All the code used to program the enemies can be found in the enemy folder, with the name corresponding to the enemy talked about in the write-up.

finalBrain.ino is the final code used to run the bot.

h5read is a script used to extract h5 file details.

The ZUMO folder is a catkin project, and in the src folder can be found the different parts of the project.

All the code used to run the simulation can be found in the mybot folder.

All code used to run the training can be found in the sumo_training folder, with training.py being the script used to execute it all.
